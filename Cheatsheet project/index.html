HTML Figure Reference

Figure tags

Tag Name Description
<figure> Figure content Represents self-contained content, typically an image with an optional caption
<figcaption> Figure caption Adds a caption or legend to the contents of the figure element
<img> Image Adds an image or picture to the contents of the figure element

Image attributes

Attribute Name Description
alt Alternative text Defines an alternative text description of the image
src Source Declares the URL of the image and is mandatory
width Width The width of the image in pixels
height Height The height of the image in pixels

Note that the <figure> and <figcaption> elements only include global attributes.
